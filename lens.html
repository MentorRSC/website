Lens Information

A concave lens is a diverging lens that spreads out light rays. It always forms a virtual, upright, and diminished image.

Image Type: Virtual
Image Orientation: Upright
Image Size: Diminished
Object Distance: 30 cm

Key Concepts

Focal Point (F): The point where parallel rays converge or appear to diverge from.

Optical Center: The center point of the lens.

Principal Axis: The horizontal line passing through the optical center.

How to Use the Lens Simulator

  • Select between concave and convex lenses using the buttons
  • Drag the object (orange rectangle) left or right to change its position
  • Adjust the focal length and object height using the sliders
  • Observe how the image (green rectangle) changes based on object position
  • Study the light rays to understand how images are formed

Learning Objectives:

  • Understand the difference between concave and convex lenses
  • Learn how image characteristics change with object position
  • Visualize the path of light rays through lenses
  • Relate focal length to image formation
